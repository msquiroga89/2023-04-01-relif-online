Configurar tu espacio de trabajo

Como en todos los talleres de Carpentries, vas a estar "codeando junto" a las instructoras. Para hacerlo, vas a necesitar tener tanto una pantalla con la herramienta que estás aprendiendo (RStudio, una consola, tu navegador, etc.) como la pantalla para la videoconferencia de Zoom. Para poder ver ambas al mismo tiempo, te recomendamos usar alguna de las siguientes opciones:

  • Dos monitores: Si tenés dos monitores, organizate para tener la herramienta sobre la que vas a aprender en un monitor y la videconferencia en el otro.
  • Dos dispositivos: Si no tenés dos monitores, ¿tenés un segundo dispositivo (tablet, smartphone) con una pantalla de tamaño mediano o grande? En ese caso, intentá usar el dispositivo más pequeño para conectarte a la conferencia y el dispositivo más grande (notebook o computadora de escritorio) para utilizar la herramienta sobre la que estás aprendiendo.
  • Dividir la pantalla: Si solo tenés un dispositivo y una pantalla, practicá tener dos ventanas (la videoconferencia y la herramienta que vas a usar) abiertas al mismo tiempo. ¿De qué forma las podés ordenar para verlas bien? ¿Te resulta más cómo cambiar de una a otra con un atajo de teclado? Probalo antes del taller para decidir qué te resulta más cómodo.
